# Messages 4 Evar

 An interface that shows a very long list of messages on a mobile device.

 ![Messages 4 Evar](/meme.png)
 
 ## Getting Started
 
 Clone the repo and then;
 
 ```
 npm install
 npm run build
 npm start
 ```
 
 Visit http://localhost:8080 in your browser.
 
 ## Testing
 
 ```
 npm test
 ```
